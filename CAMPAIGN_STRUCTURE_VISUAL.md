## 🎯 Conversion Funnel

```
AWARENESS (Top of Funnel)
│
├── Broad Match Keywords
├── Informational Queries
├── High Volume, Low Intent
│   Example: "hvað er bókhald"
│   Action: Educate, build trust
│   Landing: Blog posts, guides
│
↓
CONSIDERATION (Middle of Funnel)
│
├── Phrase Match Keywords
├── Service-Specific Queries
├── Medium Volume, Medium Intent
│   Example: "bókhaldsþjónusta reykjavík"
│   Action: Show value proposition
│   Landing: Service pages (#thjonusta, #laun)
│
↓
DECISION (Bottom of Funnel)
│
├── Exact Match Keywords
├── High-Intent Queries
├── Low Volume, HIGH Intent
│   Example: [bókhaldsstofa], [launavinnsla]
│   Action: Call-to-action (contact form, phone)
│   Landing: Contact page (#samband)
│
↓
CONVERSION
│
├── Form Submission (50.000 ISK value)
├── Phone Call >60s (40.000 ISK value)
└── Email Click (20.000 ISK value)
```

## 🔄 Weekly Optimization Workflow

```
MÁNUDAGUR (Monday)
├── Review weekend performance
├── Check Search Terms Report
└── Add negative keywords

ÞRIÐJUDAGUR (Tuesday)
├── Analyze CTR by ad group
├── Pause ads with CTR < 3%
└── Test new ad variations

MIÐVIKUDAGUR (Wednesday)
├── Review Quality Scores
├── Adjust bids on low QS keywords
└── Check landing page performance

FIMMTUDAGUR (Thursday)
├── Review conversion data
├── Analyze attribution
└── Update UTM tracking

FÖSTUDAGUR (Friday)
├── Weekly report review
├── Plan next week's tests
└── Competitor research

VIKENDI (Weekend)
├── Let campaigns run
└── Monitor for any issues
```
