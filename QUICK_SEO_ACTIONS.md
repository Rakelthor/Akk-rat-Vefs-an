### 2️⃣ Google Search Console (10 mín) - **PRIORITY!**
```bash
1. Farðu á: https://search.google.com/search-console
2. Smelltu "Add property"
3. Veldu "Domain" method
4. Sláðu inn: gloggva.is
5. Verify með DNS (bættu TXT record í domain settings)
6. Eftir verification:
   - Smelltu "Sitemaps" → Add sitemap: https://gloggva.is/sitemap.xml
   - Smelltu "URL Inspection" → Inspect: https://gloggva.is/
   - Request indexing
```
